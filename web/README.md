# K-Agenda frontend

React + TypeScript + Vite. Builds to `../out`, which `k-agenda-server.el` serves.

```sh
npm install
npm run build   # production build -> ../out
npm run dev     # dev server with HMR, connects to the same live websocket
```

See the [repo root README](../README.md) for what this is and how to install the whole package.
